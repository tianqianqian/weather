2. base64图片常用于一些图片大小较小的图片（一般图片小于10kb的时候，我们才会选择使用base64图片），减小服务器资源请求压力，若资源过大，用base64 得不偿失
  缺点： base64无法缓存

        base64格式的图片是文本格式，占用内存小，转换后的大小比例大概为1/3，降低了资源服务器的消耗；
